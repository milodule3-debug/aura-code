Chapter 1

What Is Aura Code?

Your personal AI assistant that reads, writes, and fixes code — so you don't have to be a programmer to get things done.

In Plain English

Imagine you have a super-smart friend who can look at any folder on your computer and work with its files. That's Aura. You type what you want in plain English, and Aura does it.

Aura works in a 5-step loop:

  1. Read — scans your files and understands your project
  2. Plan — decides what to change and how
  3. Execute — writes code, makes edits, runs commands
  4. Verify — runs tests, checks everything still works
  5. Report — honestly summarizes what was done with proof
Real example: You type aura "add dark mode to my website"
Aura reads your files, figures out which ones control colors, edits them, and confirms it works. All you did was type one sentence.

Built by Agents, for Everyone

Aura was built by four AI agents working together under the direction of Dusan Milosavljevic. She knows this — it's part of her origin story.

Chapter 2

Installing on Windows 11

Don't have Node.js yet? No problem. Follow these steps — takes about 10 minutes.

Step 1: Download Node.js

1

Open your web browser

Edge, Chrome, or Firefox — any browser works. You already have one!

2

Go to nodejs.org

Type nodejs.org in the address bar at the top and press Enter.

3

Click the big green LTS button

You'll see two big buttons. Click the one that says "LTS" (Long Term Support — the most stable version). A file will download.

4

Install Node.js

Double-click the downloaded node-vXX.XX.X-x64.msi file. Click Next, accept the license, keep ALL defaults, click Install. When Windows asks "allow this app to make changes?" — click Yes.

5

Finish and restart

Click Finish. It's a good idea to restart your computer now.

Step 2: Open Windows Terminal

Press the Windows key on your keyboard (it looks like a flag, between Ctrl and Alt). Type cmd or Terminal. Press Enter.

You'll see a black window with white text. This is the command line — don't be scared! You only need to type a few simple things.

Tip: You can also right-click the Start button and select "Terminal" or "Command Prompt".

Step 3: Verify Everything Works

Type these commands exactly as shown. Press Enter after each line:

C:\Users\You> node --version
v20.11.0 ← You should see a version number

C:\Users\You> npm --version
10.2.4 ← npm comes with Node.js automatically
If you see an error: Restart your computer and try again. If it still doesn't work, reinstall Node.js.

Step 4: Install Aura Code

In the same terminal, type this command and press Enter:

C:\Users\You> npm install -g aura-code

You'll see a lot of text scroll by — that's normal! Wait 1-2 minutes until the blinking cursor returns.

Now verify Aura works:

C:\Users\You> aura --version
0.6.2 ← You're ready!
Congratulations! Aura Code is now installed on your computer. Let's set up an API key so she can think.
Chapter 3

Setting Up API Keys

An API key is Aura's connection to an AI brain. Without one, she can't think. With one, she's brilliant.

What Is an API Key?

Think of an API key as a password that lets Aura talk to an AI service (like ChatGPT or Claude). It's a long string of letters and numbers that usually starts with sk-. You get one from the AI service's website, and you tell Aura to use it.

Keep your API key secret! Anyone with your key can use the AI service on your account — and you'll be billed for it. Never share it or put it in files you upload publicly.

Where to Get an API Key

Pick one — you only need one to start:

ProviderWhere to Sign UpCost
OpenAI (GPT)platform.openai.com → API keysPay-as-you-go (~$5-20/mo)
Anthropic (Claude)console.anthropic.com → API KeysPay-as-you-go (~$5-20/mo)
Google (Gemini)aistudio.google.com → Get API KeyFree tier available!
Ollama (Local)Download from ollama.comCompletely free!

How to Set Your API Key

Temporary method (lasts until you close Terminal):

C:\Users\You> set ANTHROPIC_API_KEY=sk-ant-api03-your-key-here

Or for OpenAI:

C:\Users\You> set OPENAI_API_KEY=sk-proj-your-key-here

Permanent method (survives restarts):

  1. Press Windows key, type environment
  2. Click "Edit the system environment variables"
  3. Click "Environment Variables..."
  4. Under "User variables", click "New..."
  5. Variable name: ANTHROPIC_API_KEY (or OPENAI_API_KEY)
  6. Variable value: paste your actual key
  7. Click OK on all windows. Restart Terminal.
Done! Now Aura can use whichever AI brain you chose. Test it: aura --models to see available models.

Chapter 5

Aura's Modes

Aura has different personalities for different jobs. Pick the right mode for your task.

Normal Mode (Default)

Best for: everyday tasks — fixing bugs, adding features, asking questions.

aura "fix the broken login button"

One agent. Read → Plan → Execute → Verify → Report. Simple and fast.

Orchestrate Mode

Best for: big, complex tasks that need multiple specialists.

aura --orchestrate "add error handling to every endpoint in my API"

Aura breaks your task into sub-tasks and assigns each to a specialist agent — Researcher, Coder, Reviewer — working together like a tiny software team.

Architect Mode

Best for: planning before you build. No code is written — just a blueprint.

aura --architect "design a new authentication system"

Outputs: which files to create, interfaces, dependencies, risks. "If you cannot explain why a file needs to exist in one sentence — it does not need to exist."

Verify Mode

Best for: making sure your changes didn't break anything.

aura --verify --test-command "npm test" "fix the failing tests"

After making changes, Aura runs your tests automatically. If tests fail, she fixes the issue and tries again.

Analyze Mode

Best for: learning from past mistakes.

aura --analyze

Scans your session history for recurring failure patterns and suggests how to improve.

Safety & Permissions

Aura respects safety boundaries. You're always in control.

Permission Modes

ModeWhat Aura Can Do
read-onlyOnly read files. Cannot write, edit, or run commands. Safest for exploration.
normalRead and write files, but asks you before running shell commands or making destructive changes.
autoFull autonomy. No confirmation prompts. Use with caution on projects you trust.

Aura starts in normal mode by default. She will always ask before doing anything that could be risky. You can say no.

Chapter 8

Frequently Asked Questions

The answers to questions everyone asks.

Do I need to know how to code?

No! Aura is designed to help both beginners and experts. You describe what you want in plain English. Aura figures out the technical details.

Is Aura free?

Aura Code itself is 100% free and open-source (MIT license). The AI models she uses (Claude, GPT, etc.) charge for API access — typically a few dollars per month. You can also use completely free local models via Ollama.

What can Aura NOT do?

Aura works with text files and code. She can't edit images, edit videos, control your mouse, or browse the web like a human (though she can fetch web pages). She's a coding agent — she excels at reading, writing, and fixing code.

Is my code safe with Aura?

Yes. Aura runs on your computer. Your code is sent to the AI provider's servers for processing (like ChatGPT), but Aura doesn't store or share your code. Use Ollama with local models for complete privacy.

What if Aura makes a mistake?

Aura's motto is "I don't try. I verify." She runs tests after making changes. But AI can still make mistakes — always review important changes. Use git to track changes so you can undo anything.

How do I update Aura?

npm update -g aura-code

Run this in Terminal anytime. Check version with aura --version.

Can I use Aura for non-coding tasks?

Absolutely! Aura has tools for web search, email, calendar, Telegram, WhatsApp, YouTube transcripts, audio transcription, and more. Try: aura "search the web for latest AI news"
